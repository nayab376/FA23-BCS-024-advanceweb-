# Node.js CRUD App (SQLite, MySQL, MongoDB)

## Setup

1. Run `npm install` to install dependencies.
2. Start MySQL via XAMPP.
3. Start MongoDB service (`mongod`).
4. Run `node server.js`.

## Usage

Open browser: http://localhost:3000  
Click on SQLite, MySQL, or MongoDB box → CRUD interface will open.  

## Notes

- SQLite DB (`mydb.sqlite`) is automatically created by Node.js.  
- MySQL requires `testdb` with `users` table (id, name, email).  
- MongoDB database `testdb` will be created automatically.
